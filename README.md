## 📌 About
IRA is a predictive user behavior intelligence system that tracks
user messages over time, calculates risk trajectories, and
automatically warns or bans high-risk users before harm happens.

## ✨ Features
- 🧬 Predictive risk scoring per user
- 📈 Behavior trajectory tracking (rising/stable/falling)
- 🚫 Auto-ban system for high risk users
- ⚠️ Progressive warning system
- 🎯 Live intelligence dashboard
- 💬 Chat simulator for testing

## 🛠️ Tech Stack
| Layer | Technology |
|---|---|
| Backend | Python, Flask |
| AI Model | Llama 3.3 70B |
| AI Provider | Groq API |
| Frontend | HTML, CSS, Vanilla JS |
| Deployment | Railway |

## ⚙️ Setup
1. Clone: `git clone https://github.com/lasya0809/ira`
2. Install: `pip install -r requirements.txt`
3. Set key: `export GROQ_API_KEY=your_key_here`
4. Run: `python app.py`
5. Open: `http://localhost:5000`
